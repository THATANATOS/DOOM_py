# DOOM_py
A 100% pure Python and Pygame implementation of a pseudo-3D First Person Shooter. This project demonstrates the core logic behind the DDA (Digital Differential Analyzer) raycasting technique used in classic 90s titles like Wolfenstein 3D and DOOM.
